# api-tester

### Tests a HTTP service endpoint response against expected JSON

JS Libraries used:

* knockout.js 
    - For 2-way data binding
* require.js, text.js
    - Asynchronous module loading & Modular development
* jquery 
    - DOM manipulation
* mockjax (jquery plugin) 
    - Mocking endpoint reponse
* Lodash 
    - Utility
